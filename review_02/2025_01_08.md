    2. 상속

3. Git 명령어 복습
git init : git 초기화
git add 파일명 : 이 파일을 git으로 관리하겠다. (스테이징)
git remote add origin "Github 주소" : "Github 주소를 origin이란 약어로 대체하겠다.
git commit -m main
git switch "브랜치명" : 이 브랜치에서 수정을 진행하겠다.
